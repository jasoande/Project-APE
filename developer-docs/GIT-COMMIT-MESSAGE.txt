Release v3.2.0: Simplified Dependencies & Enhanced User Experience

Major Changes:
- Removed 7 unnecessary dependencies (33% reduction: 15 → 10 packages)
- Disabled AI orchestration (standard retry logic is sufficient)
- Updated all version numbers to 3.2.0 across documentation and code
- Added King Kong logo branding to all major documentation
- Fixed message consistency (./launch_ape.sh as primary command)
- Added --help support to launch_ape.sh

Dependencies Removed:
- google-genai (AI orchestration - feature disabled, not needed)
- anthropic[vertex] (industry detection now manual in vars.py)
- reportlab (not imported, not used)
- python-dateutil (not imported, stdlib sufficient)
- notebooklm-py (CLI used via subprocess, not Python SDK)
- requests (transitive dependency, auto-installs)
- requests-oauthlib (transitive dependency)

Core Dependencies Retained (10 packages):
- Google Drive API (5 packages) - document download
- Flask + werkzeug (2 packages) - dashboard
- pypdf + Pillow (2 packages) - PDF/image processing
- python-dotenv (1 package) - configuration

Configuration Updates:
- vars.py: Disabled GEMINI_AGENT_CONFIG (enabled: False)
- Documented why AI features disabled (minimal value, added complexity)
- Standard retry logic achieves 95-98% success vs 99% with AI

Documentation Enhancements:
- Added King Kong logo to 11+ major documents (README, QUICKSTART, etc.)
- Updated all version numbers to 3.2.0 for consistency
- Added "Python Dependencies" section to README highlighting minimal deps
- Fixed message consistency - always recommend ./launch_ape.sh for local use
- Clarified advanced options (direct python3 main.py usage)
- Added context labels: "Container:", "Advanced:" to reduce confusion

User Experience Improvements:
- Added ./launch_ape.sh --help (and -h) with comprehensive usage examples
- Clarified "Disable dashboard" section in QUICKSTART (Advanced option)
- Consistent messaging across all automation scripts
- Clear separation: local (./launch_ape.sh) vs container (python3 main.py) vs advanced

Code Changes:
- main.py: Updated version banner to "3.2.0 - Simplified Dependencies Release"
- launch_ape.sh: Added help text with usage examples, modes, and dashboard info
