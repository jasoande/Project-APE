## Quick Build Instructions

1. **Create Presentation**
   - Go to slides.google.com
   - Click "Blank" presentation
   - Name: "Project APE - Management Demo"

2. **Set Theme**
   - Slide → Edit theme
   - Background: Gradient (navy #1a1f3a to black)
   - Default fonts: Montserrat (headings), Open Sans (body)

3. **Add Logo**
   - Upload: `dashboard/static/kingkong.png`
   - Slide 1: Center, large
   - Slides 2-10: Top-right, small

4. **Copy Content**
   - Copy each slide's content above
   - Paste into slides
   - Format with specified fonts/colors

5. **Add Animations** (Optional)
   - Keep subtle: Fade-in only
   - Slide 7: Numbers count up
   - Slide 8: Cards appear left-to-right
