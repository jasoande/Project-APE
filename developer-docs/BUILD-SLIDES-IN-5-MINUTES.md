# Build Your Project APE Presentation in 5 Minutes
## Super Quick Guide - Google Slides

---

## Step 1: Create Presentation (30 seconds)

1. Go to **slides.google.com**
2. Click **"Blank"** presentation
3. Name it: **"Project APE - Management Demo"**

---

## Step 2: Set Dark Theme (1 minute)

1. Click **Slide** → **Edit theme**
2. Click the background box
3. Choose **Gradient**:
   - Color 1: `#1a1f3a` (navy blue)
   - Color 2: `#0a0e1a` (black)
   - Direction: Top to bottom
4. Set default fonts:
   - Title: **Montserrat Bold, 48pt, White**
   - Body: **Open Sans, 18pt, White**
5. Click **Done**

---

## Step 3: Add King Kong Logo (1 minute)

1. On **Slide 1** (title slide):
   - Click **Insert** → **Image** → **Upload from computer**
   - Upload: `dashboard/static/kingkong.png`
   - Resize: Make it BIG (60% of slide height)
   - Center it

2. For **Slides 2-10**:
   - Go to **Slide** → **Edit theme**
   - Insert logo in top-right corner (small, 100px)
   - This adds it to ALL slides automatically
   - Click **Done**

---

## Step 4: Copy-Paste Content (2 minutes)

Open **`SLIDES-CONTENT-READY-TO-USE.md`** and:

1. **For each slide:**
   - Copy the content under "Content to Add"
   - Paste into your Google Slide
   - Format with the fonts/colors shown

2. **Quick formatting tips:**
   - Headlines: Montserrat Bold, 48pt, White
   - Metrics/Numbers: Roboto Mono, 32pt, Gold `#d4af37`
   - Body text: Open Sans, 18pt, White
   - Bullets: Use emojis from the content

---

## Step 5: Add Footer (30 seconds)

1. Go to any slide 2-10
2. Click **Slide** → **Edit theme**
3. Add footer text boxes:
   - **Left:** `Slide #`
   - **Center:** `Project APE v3.1.0`
   - **Right:** `Production Release`
4. Font: Open Sans, 10pt, Gray
5. Click **Done**

---

## DONE! ✅

**You now have a professional presentation ready to present.**

## Before Presenting Checklist

- [ ] Logo looks good on all slides
- [ ] Text is readable (not too small)
- [ ] Colors match theme (navy, gold, white)
- [ ] Footer on slides 2-10
- [ ] Presenter notes printed/accessible
- [ ] Demo ready (dashboard + NotebookLM open)
- [ ] Backup screenshots in case demo fails

---

## Pro Tips

### For Slide 1 (Title)
- Make King Kong logo HUGE
- Text below logo in gold/white
- Keep it simple and dramatic

### For Slide 6 (Demo)
- Have dashboard pre-loaded in browser
- Have NotebookLM notebook open in another tab
- Close all other tabs/apps
- Zoom browser to 125% so everyone can see

### For Slide 7 (ROI)
- Make the numbers BIG
- Use gold color for savings amount
- Consider adding a simple bar chart

### For Slide 8 (Pilot Results)
- Use cards/boxes for each success story
- Green checkmarks for "Quality: 8.7/10"
- Make "100% SUCCESS" huge at bottom

### For Slide 10 (Rollout)
- Use timeline visualization if possible
- Highlight "What We Need" in gold box
- Make "APPROVE" the biggest word on slide

---

## Time-Saving Shortcuts

**If you're REALLY short on time:**

1. Use a **dark template** instead of custom theme
2. Just paste content as-is (format later)
3. Logo only on title slide (skip theme editing)
4. Skip footer (nice-to-have, not critical)
5. Focus on Slides 1, 2, 6, 7, 10 (most important)

**Minimum viable presentation:** 5 slides
- Slide 1: Title
- Slide 2: Problem
- Slide 6: Demo
- Slide 7: ROI
- Slide 10: Rollout Plan

## Test Your Presentation

1. **Presenter View:**
   - Click **View** → **Presenter view**
   - Check that you can see notes
   - Practice advancing slides

2. **Timing:**
   - Run through once with timer
   - Target: 45 minutes + 15 min Q&A
   - Adjust if over/under

3. **Demo:**
   - Actually run the demo
   - Make sure dashboard loads
   - Make sure NotebookLM opens
   - Prepare backup screenshots

---

## Export Backup

**Just in case tech fails:**

1. **PDF Backup:**
   - File → Download → PDF
   - Have on your computer

2. **Print Key Slides:**
   - Print slides 6-8 (demo, ROI, results)
   - If projector fails, you can talk through them
