# How to Create the Project APE Google Slides Presentation

## Quick Start

1. Go to [Google Slides](https://slides.google.com)
2. Create new presentation: **Blank**
3. Follow the instructions below for each slide

---

## Theme Setup (Do This First)

### Set Master Theme

1. Click **Slide** → **Edit theme**
2. Set background:
   - **Type:** Gradient
   - **Colors:** #1a1f3a (navy) to #0a0e1a (black)
   - **Direction:** Top to bottom
3. Set default fonts:
   - **Headings:** Montserrat Bold, 48pt, White (#ffffff)
   - **Subheadings:** Montserrat SemiBold, 32pt, Gold (#d4af37)
   - **Body:** Open Sans, 18pt, White (#ffffff)
4. Click **Close** to save theme

### Add Logo to Master

1. In theme editor, add logo to top-right corner (small, 100px width)
2. Add footer:
   - Left: Slide number
   - Center: "Project APE v3.1.0"
   - Right: Your name

## Final Touches

### Before Presenting

1. **Add slide numbers** to all slides
2. **Check transitions**: Use "Fade" (subtle, professional)
3. **Test animations**: Don't overdo it
4. **Proofread**: No typos allowed
5. **Export PDF backup**: File → Download → PDF
6. **Test on presenting computer**: Fonts render correctly?
7. **Practice with clicker**: Know which button advances

### Optional Enhancements

- Add Red Hat logo in top corner of all slides
- Use Red Hat brand colors if presenting internally
